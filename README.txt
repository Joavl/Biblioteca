Um projeto de Biblioteca usando Console Application

Usando conceitos de Orientação a Objetos , funções e collections .

Principais funcionalidades : 

*Cadastro de Livros
*Pesquisa de Livros por ID
*Listagem de Livros
*Exclusão de Livros
*Empréstimos de Livros
*Devolução de Livros

*Cadastro de Usuários
*Listagem de Usuários
*Exclusão de Usuários
